## Project 1: Hotel Management System

### Description
A comprehensive database-driven hotel management solution designed to streamline operations and improve guest services.

### Technologies
- **Backend**: Python
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Architecture**: Relational Database with Python scripting

### Key Features
- **Guest Management**: Complete guest profile management with booking history
- **Room Management**: Real-time room availability tracking and allocation
- **Booking System**: Automated booking and reservation processing
- **Payment Processing**: Integrated payment tracking and billing
- **Multi-User Access**: Role-based access control (Admin, Staff, Guest)
- **Reporting**: Generate occupancy and revenue reports

### Database Design
The system uses MySQL with normalized database schema including:
- `guests` table - Guest information and contact details
- `rooms` table - Room details, type, and pricing
- `bookings` table - Reservation records with dates
- `payments` table - Transaction history and receipts

### Skills Demonstrated
- Database normalization and design
- SQL query optimization
- Python database connectivity
- Business logic implementation
- User authentication and authorization

### Repository
[B103_Hotel_Database_Project](https://github.com/pratikshingala03/B103_Hotel_Database_Project)

## Project 2: Enterprise Restaurant Management System (ERMS)

### Description
A large-scale, multi-branch restaurant management platform designed for enterprise-level operations with sophisticated inventory, order, and staff management capabilities.

### Technologies
- **Backend**: Node.js with Express.js
- **Language**: TypeScript
- **Database**: MongoDB
- **Frontend**: React with TypeScript
- **Architecture**: Microservices-ready design

### Key Features
- **Multi-Branch Support**: Centralized management for multiple restaurant locations
- **Inventory Management**: Real-time stock tracking and automated reordering
- **Order Processing**: Kitchen display system and order routing
- **Staff Management**: Scheduling, performance tracking, and payroll integration
- **Analytics Dashboard**: Real-time business metrics and insights
- **Payment Integration**: Multiple payment method support
- **Customer Management**: Loyalty programs and customer preferences

### Database Schema
MongoDB collections include:
- `restaurants` - Branch/location information
- `inventory` - Stock levels and supplier data
- `orders` - Order details with timestamps
- `staff` - Employee information and schedules
- `customers` - Customer profiles and preferences

### Skills Demonstrated
- Full-stack TypeScript development
- NoSQL database design with MongoDB
- Real-time data synchronization
- Business analytics and reporting
- Scalable architecture design
- API design and RESTful principles

### Repository
[Enterprise-Restaurant-Management-System-ERMS-](https://github.com/pratikshingala03/Enterprise-Restaurant-Management-System-ERMS-)
